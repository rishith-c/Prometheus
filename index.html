Prometheus

An autonomous GPS quadcopter that flies waypoint missions, holds position, and returns to launch on its own. Flown from a Raspberry Pi over MAVLink.

View the project on GitHub

Overview

Prometheus is a 1.85 kg quadcopter I designed and built from scratch. The goal is a drone that can fly a set of GPS waypoints, hold its position, and come home by itself, with a Raspberry Pi running the mission code and talking to the flight controller over MAVLink. Everything needed to build and program it lives in the repo: hardware specs, wiring, the ArduCopter config, and the Python flight software.

What it can do

  • GPS waypoint navigation using DroneKit-Python
  • MAVLink control from a Raspberry Pi 4 companion computer
  • GPS position hold and altitude hold
  • Automatic return-to-launch on low battery or lost signal
  • 12 to 15 minutes of hover on a 3000mAh 4S pack
  • Ready for a computer vision payload later on

Where it's at

How it's put together

The Raspberry Pi runs the Python mission code and sends MAVLink commands to the Matek flight controller. The flight controller handles the sensor fusion from the GPS, IMU, and barometer, and drives the four motors through the ESC. On low battery or a lost link, it returns to launch on its own.
